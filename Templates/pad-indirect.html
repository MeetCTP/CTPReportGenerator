Report Description

The PA Distance Indirect Time Report is used to check if a provider has used more indirect time than the school allows. The appointments shown in the report will be between the Start Date and End Date the user provides. It details how much direct and indirect time each provider/client pair have used along with how much indirect time is allowed and whether the provider exceeded that limit or not.
Some Central Reach labels may be inaccurate due to duplicates in their system.
