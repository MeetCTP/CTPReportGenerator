Report Description

The High Client Cancellation Report details a list of appointments within a 3 month timeframe. It then groups all sessions by provider/client pairings and calculates if each provider/client pair has had 3 cancallations in a row or not. It also calculates the percentage of sessions that were cancelled. All these calculations are broken down month to month. Currently, this report is rigid in its execution method. The start date should always be determined by taking the current month, going back two months, and choosing the first of that month. For Example: If the current month is May, the start date should be the first of the month, two months prior, making it March 1st.
Some Central Reach labels may be inaccurate due to duplicates in their system.
